# [0050. Pow(x, n)](https://leetcode.cn/problems/powx-n/)

- 标签：
- 难度：中等

## 一、题目说明

**描述**

实现 [pow(*x*, *n*)](https://www.cplusplus.com/reference/valarray/pow/) ，即计算 `x` 的整数 `n` 次幂函数（即，`xn` ）。

**示例**

* 示例 1：

```text
输入：x = 2.00000, n = 10
输出：1024.00000
```

* 示例2：

```text
输入：x = 2.10000, n = 3
输出：9.26100
```

* 示例3：

```text
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
```

**提示**

* `-100.0 < x < 100.0`
* $-2^{31} <= n <= 2^{31}-1$
* `n` 是一个整数
* $-10^4 <= x^n <= 10^4$
