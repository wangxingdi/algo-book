# 0032. [最长有效括号](https://leetcode.cn/problems/longest-valid-parentheses/)

* 标签：

* 难度：困难

## 一、题目说明

**描述**

给你一个只包含 `'('` 和 `')'` 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

**示例**

* 示例 1：

```text
输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"
```

* 示例2：

```text
输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"
```

* 示例3：

```text
输入：s = ""
输出：0
```

**提示**

* $0 <= s.length <= 3 * 10^4$

* `s[i] 为 '(' 或 ')'`
