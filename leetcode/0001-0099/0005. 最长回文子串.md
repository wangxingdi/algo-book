# [0005. 最长回文子串](https://leetcode.cn/problems/longest-palindromic-substring/)

* 标签：

* 难度：中等

## 一、题目说明

**描述**：给你一个字符串`s`，找到`s`中最长的回文子串。

**说明**：如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。

**示例**

* 示例 1：

```text
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```

* 示例 2：

```text
输入：s = "cbbd"
输出："bb"
```

**提示**

* $1 <= s.length <= 1000$

* `s` 仅由数字和英文字母组成
